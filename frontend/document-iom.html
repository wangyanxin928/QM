IOM介绍

IOM(Interllignce Organizational Management)，智能权限管理：
        企业的组织结构是一个企业基本的构成形态。现代化的企业管理需要借助管理软件实现科学化规范化的自动管理。 IOM_V1.5集成了对企业的基本组织架构管理，机构与员工的管理，职位体系的管理，职位体系与员工的管理，服务、APP与用户的管理等； iOM中还设置了对云中应用权限的管理机制；从而实现企业组织机构与云中应用的管理一体化管理。

接口调用标准及规范

IOM提供对外开放接口，可以查询当前私有云的组织机构树、结构下人员、角色信息，同时可以获取角色或人员拥有的应用及其他权限信息。

公有云接口地址：

目前系统提供两种接口调用方式：


                    WebSerivces：http://www.gnetcloud.com/Gnet_Ibd_Iom/services/IOM_CommonService?wsdl

                    Restful：http://www.gnetcloud.com/Gnet_Ibd_Iom/services/IOM_CommonRest?_wadl

调用规范：

以获取机构下人员接口为例
示例代码： 


                    [
                    {
                    "IOM_Version": "Web3.0"
                    },
                    {
                    "organId": "a8ff3ac2f0704188839e3f0f698d560b",
                    "pageSize": "10000",
                    "pageIndex": "1",
                    "serviceId": "IBD104"
                    }
                    ]
