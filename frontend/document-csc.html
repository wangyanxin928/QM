社区介绍

CSC(Community Service Center)，社区服务中心：
        社区集成了云社区、三大商店及开发者社区三大模块；所有在社区注册的私有云均可以在云社区中看到，Gnet账户可以选择感兴趣的私有云申请加入；三大商店即将应用、数据、服务于一体，用户可以在商店中找到自己感兴趣的APP，购买下载到IBD桌面使用；开发者社区，为开发者在开发应用是提供帮助说明。
        同时社区提供接口供第三方开发者使用，例如登陆、注册、加入/退出私有云、获取应用等。

社区访问地址

社区官网访问地址： http://www.gnetcloud.com

接口调用标准及规范

社区对外开发部分接口，供开发者在开发应用时使用，可以实现登陆、注册及对人员管理等功能。

接口地址：

目前系统提供两种接口调用方式：


                    WebSerivces：http://www.gnetcloud.com/Gnet_Ibd_Iom/services/Ibd_CommonService?wsdl

                    Restful：http://www.gnetcloud.com/Gnet_Ibd_Iom/services/Ibd_CommonRest?_wadl

调用规范：

接口调用参数格式： 


                    {
                    "GII_HEAD": {  //头信息
                    "VERSION": "",  //端类型及版本号，例如电脑（windows系统）：PC1.0
                    "SNID": "",  //时间戳
                    "SIGN": [
                    {
                    "position/k": ""
                    },
                    {
                    "RandomCode/v": ""
                    }
                    ],
                    "GZIP": "false",
                    "CLOUD_ID": ""   //调用私有云id
                    },
                    "GII_IBD": {}   //参数信息
                    }

注意：
