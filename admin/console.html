云API APICloud开发的应用安全性怎么样？

完全可以使用自己的服务器。APICloud通过规范化的封装IOS和Android平台的各种功能，并以JS对象的形式统一开放端API给开发者调用。你可以将APICloud理解为使用HTML + CSS + JS + API进行移动应用开发，它的开发流程跟通常的Android、IOS等App的开发流程一致，如果需要访问网络服务器，可以使用端API中网络相关的API向你约定的任何服务器进行数据请求。

APICloud在整个开发流程的各个环节中为开发者提供便利以及提升效率，同时还提供App上线运营的一些基础服务，如PUSH、统计分析、App编译、App版本管理、云存储等等。

你也可以使用APICloud提供的云存储服务，将数据库部署在APICloud云上，通过端API中MCM模块下的各个API进行增删改查以及文件、用户管理等数据操作，这些操作均符合restful标准，你也可以通过使用restful接口形式进行数据访问（相当于NoSQL）。可减少大量的服务器编码以及运维工作。
